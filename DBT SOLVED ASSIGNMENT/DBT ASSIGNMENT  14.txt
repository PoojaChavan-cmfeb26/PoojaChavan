1) Write a command that puts the following values, in their given order, into the 
salespeople table: city – San Jose, name – Blanco, comm – NULL, cnum – 1100.

INSERT INTO salespeople (city, name, comm, cnum)
VALUES ('San Jose', 'Blanco', NULL, 1100);

2) Write a command that removes all orders from customer Clemens from the 
Orders table. 

DELETE FROM Orders
WHERE cnum = (
    SELECT cnum
    FROM Customers
    WHERE cname = 'Clemens'
);


3) Write a command that increases the rating of all customer in Rome by 100. 

UPDATE Customer
SET rating = rating + 100
WHERE city = 'Rome';




4) Salesperson Serres has left the company. Assign her customers to Motika.

UPDATE Customers
SET snum = (
    SELECT snum
    FROM Salespeople
    WHERE name = 'Motika'
)
WHERE snum = (
    SELECT snum
    FROM Salespeople
    WHERE name = 'Serres'
);
