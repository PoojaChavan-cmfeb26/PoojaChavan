Fix infinite loop: while(i < 10){ System.out.println(i); }
That loop runs infinitely because the value of i never changes, so the condition i < 10 is always true.

=>


public class FixInfiniteLoop {
    public static void main(String[] args) {
        int i = 0; // initialize i

        while (i < 10) {
            System.out.println(i);
            i++; // increment i to avoid infinite loop
        }
    }
}
